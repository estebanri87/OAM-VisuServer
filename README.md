# OpenKNX VisuServer

Der OpenKNX VisuServer stellt eine moderne Web-Visualisierung fuer KNX-Anlagen bereit.

## Features

- Responsive Web-Visualisierung fuer Smartphone, Tablet und Desktop
- Live-Updates ueber WebSocket
- Dashboard mit Favoriten sowie strukturierte Raum- und Etagenansicht
- Konfiguration der Visualisierung direkt ueber ETS
- Konfigurierbar über ETS

### Unterstuetzte Visualisierungselemente

- Schalten
- Dimmen
- Jalousie / Beschattung
- Thermostat
- Szene
- RGB
- Wertanzeige
- Kamera
- Taster
- Musik

Eine vorkomplierte Firmware ist [hier](https://github.com/estebanri87/OAM-VisuServer/releases) zu finden. ZIP Datei herunterladen, entpacken und der Anleitung im Readme folgen.

## Hardware

Als Hardware kann jede OpenKNX oder OpenKNX-Ready Hardware mit LAN oder WLAN verwendet werden.
Die vorkompilierte Firmware unterstützt:

- [REG1-LAN-TP-Base](http://device.openknx.de/REG1-LAN-TP-Base)
