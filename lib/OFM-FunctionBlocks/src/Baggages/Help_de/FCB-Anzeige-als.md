### Anzeige als

Gibt an, wie der Bitwert angezeigt werden soll.

- Textbausteine aus 'Format Optionen'
  Es werden die unter 'Textbaustein Aus' bzw. 'Textbaustein Ein' definierten Texte verwendet.
- 0 / 1
- Aus / Ein
- Inaktiv / Aktiv
- OK / Fehler
- Normal / Störung
- Bereit / Alarm
- Geschlossen / Offen
- Zu / Auf
- Versperrt / Entsperrt
- Verriegelt / Entriegelt
- Abwesend / Anwesend
